Standard Units
==============

TAZ uses the following standard units throughout the code.

| Value          | Units |
|:-------------- |:-----:|

| Nuclear Radius | fm    |
| Channel Radius | fm    |
| Energy         | eV    |
| Partial Widths | eV    |
